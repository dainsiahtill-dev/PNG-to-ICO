# PNG to ICO Converter 🖼️➡️📂

一个基于 **Electron** + **Vue 3** 构建的轻量级图片转换工具。旨在提供最简单、快捷的方式将 PNG 图片转换为 ICO 图标文件。

## ✨ 功能特点

* **极简操作**：没有复杂的设置，专注于转换核心功能。
* **拖拽支持**：直接将 PNG 图片拖入应用窗口即可开始转换。
* **即时生成**：快速生成标准的 `.ico` 文件。
* **现代技术栈**：使用 Vue 3 + TypeScript + Vite 开发，体验流畅。

## 🛠️ 技术栈

本项目基于以下技术构建：

* [Vue 3](https://vuejs.org/) - 前端框架
* [Vite](https://vitejs.dev/) - 构建工具
* [TypeScript](https://www.typescriptlang.org/) - 类型安全
* [Electron](https://www.electronjs.org/) - 跨平台桌面应用框架
